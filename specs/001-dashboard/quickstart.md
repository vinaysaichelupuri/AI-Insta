# Quickstart: Dashboard

## Prerequisites
- Node.js 18+
- MongoDB
- Instagram Professional Account credentials

## Setup
1. Clone the repository.
2. Run `npm install` in the root directory to bootstrap the monorepo workspaces.
3. Configure environment variables for the frontend and backend (MongoDB URL, Gemini API Key, Instagram API Key).
4. Run `npm run dev` from the root to start both backend and frontend servers.
